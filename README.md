# DuMessengerLAN

DuMessengerLAN is a LAN-based messaging application consisting of two components: DuMessengerServer and DuMessengerClient.

## DuMessengerServer

DuMessengerServer is a messaging server that runs on a local network. It listens for connections from clients and manages communication between connected clients.

### Usage

To run the DuMessengerServer application, follow these steps:

1. Open QT Creator.
2. Open the DuMessengerServer.pro project file.
3. Build and run the project.
4. The server will start listening on the specified port.

## DuMessengerClient

DuMessengerClient is a messaging client that connects to the DuMessengerServer running on the local network. Once connected, the user can send messages.

### Usage

To run the DuMessengerClient application, follow these steps:

1. Open QT Creator.
2. Open the DuMessengerClient.pro project file.
3. Build and run the project.
4. Enter the IP address and port of the DuMessengerServer you want to connect to.
5. Click the "Connect" button to establish a connection to the server.
6. Enter your message in the text box and click the "Send" button to send the message.
